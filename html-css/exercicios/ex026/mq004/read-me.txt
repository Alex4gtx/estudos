media query = media type + media features

mobile first:
    conceito de criação de site que aborda uma ideia de criar um site a partir de um site versão mobile,
    esse conceito se difere do conceito antigo que primeiro era feito para computador e depois eram retirados
    alguns conteúdos para se adaptar com os dispositivos mobile, esse conceito antigo além de mais trabalhoso,
    fazia com que os sites mobiles acabassem tendo poucas features, já com o novo conceito se torna mais fácil
    e também se enquadra em boas praticas, fazendo com que os motores de buscas deem um crédito a mais em sites
    construidos desta forma.

device breakpoints:
    typical devices breakpoints
    celular: telas de até 600px
    tablet: de 600px a 768px
    desktop: 992px até 1200px
    grandes telas: acima de 1200px

    novo:
        320px — 480px: dispositivos móveis.
        481px — 768px: iPads, tablets.
        769px — 1024px: telas pequenas, laptops.
        1025px — 1200px: desktops, telas grandes.
        1201px e acima — telas muito grandes, TVs.
